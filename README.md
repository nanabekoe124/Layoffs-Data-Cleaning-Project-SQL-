# Layoffs-Data-Cleaning-Project-SQL-
"SQL data cleaning project on layoffs data using CTEs, window functions, updates, duplicate removal, and data standardization."
"This project focuses on cleaning and preparing a layoffs dataset for analysis using SQL.

Key Tasks Performed
Removed duplicate records using Common Table Expressions (CTEs) and window functions.
Standardized inconsistent data entries, including company names, industries, and country fields.
Handled null and missing values where appropriate.
Updated records to improve data consistency and accuracy.
Formatted date fields into a standardized structure for easier analysis.
Validated data quality to ensure reliable reporting and downstream analysis.
SQL Concepts Used
Common Table Expressions (CTEs)
UPDATE Statements
DELETE Statements
Window Functions (ROW_NUMBER)
String Functions (TRIM, UPPER, LOWER)
Data Standardization Techniques
Null Value Handling
Objective

The goal of this project was to transform raw, inconsistent layoffs data into a clean and analysis-ready dataset that can be used for my personal business intelligence, reporting, and exploratory data analysis."
